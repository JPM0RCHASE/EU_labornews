# 카드뉴스·뉴스레터 가시성·매력도 검토

> 목표: 일 방문자 300~500명 (수익화는 이후 단계)
> 검토 범위: ① 가시성(네이버 검색노출) ② 매력도(클릭 유도) — 두 가지만

---

## 한 줄 진단

> **"썸네일은 좋아졌는데, 본문이 '링크 한 줄'이라 매력도·검색노출 둘 다 손해 보고 있다."**

지금 블로그 글을 열면 보이는 것:
```
[썸네일 이미지]
https://eu-labornews.vercel.app/.../labornews_20260629.html   ← 링크 한 줄
#해시태그 ...
```

→ 독자는 **블로그 안에서 볼 게 없어서** vercel로 나가야 함 = 이탈
→ 네이버는 **색인할 글자가 없어서** "저품질 문서"로 판단 = 검색노출 ↓

---

## 1. 가시성 (네이버 검색노출) 검토

네이버 블로그 상위노출의 3대 요소: **제목 키워드 · 본문 텍스트량 · 체류시간**

### ❌ 문제 1 — 제목에 날짜코드가 앞에 있음
```
현재:  "260629 금일의 HR이슈 카드뉴스"
```
- `260629`는 검색이 안 되는 숫자 → 제목 앞자리 낭비
- "금일의 HR이슈"는 아무도 검색 안 하는 표현
- 정작 오늘의 핵심 키워드(노란봉투법, 현대차 파업)는 제목에 없음

```
개선:  "노란봉투법·현대차 파업 ｜ 2026.6.29 오늘의 노동뉴스 5선"
```
→ 사람들이 실제로 검색하는 단어를 **제목 앞쪽**에 배치

### ❌ 문제 2 — 본문에 텍스트가 없음 (가장 치명적)
네이버는 글자 수 적은 링크글을 거의 노출 안 시켜요.
지금은 본문 텍스트가 사실상 0줄.

→ **해결: 5개 뉴스 제목 + 한 줄 요약을 본문에 넣기**
   (이미 카드뉴스 데이터에 다 있는 내용 → 자동으로 본문에 풀어주면 됨)

### ❌ 문제 3 — 뉴스레터 글은 이미지조차 없음
카드뉴스는 썸네일 들어감(✅ 잘함). 근데 뉴스레터 글(6월 5째주)은 이미지 0.
→ 목록에서 회색 빈칸 = 클릭률 바닥

---

## 2. 매력도 (클릭하고 싶은 정도) 검토

독자가 글 목록에서 보는 건 딱 3가지: **썸네일 · 제목 · 첫 줄 미리보기**

### ✅ 잘하고 있는 것
- 카드뉴스 썸네일(1200×630, 헤드라인 3줄) — 이건 정말 좋아졌어요. 유지!
- 해시태그 풍부함 — 관련도 높은 태그만 잘 들어감

### ❌ 아쉬운 것 — 첫 줄 미리보기가 'URL'임
네이버 목록은 본문 첫 줄을 미리보기로 보여줘요.
지금 카드뉴스 첫 줄 = `https://eu-labornews...` (링크)
→ 목록에서 **파란 URL만 덩그러니** = 매력 0

```
개선 — 첫 줄을 후킹 문구로:
"오늘 노동 뉴스 핵심만 3분 정리 📌 노란봉투법·현대차 파업·금융권 교섭"
```

### ❌ 아쉬운 것 — 본문에 '읽을 거리'가 없음
매력도 = "이 안에 내용이 알차 보인다"는 느낌.
링크만 있으면 "광고글" 느낌이 나서 오히려 이탈해요.
→ 5줄 요약만 본문에 있어도 "정성 들인 글"로 보임

## 3. 우선순위 (가시성·매력도만 기준)

| 순위 | 개선 | 가시성 | 매력도 | 자동화 |
|------|------|:------:|:------:|:------:|
| 🔴 1 | 본문에 뉴스 5줄(제목+한줄요약) 넣기 | ◎ | ◎ | 가능 |
| 🟠 2 | 제목을 키워드 앞으로 (날짜코드 제거) | ◎ | ○ | 가능 |
| 🟡 3 | 첫 줄을 후킹 문구로 | ○ | ◎ | 가능 |
| 🟢 4 | 뉴스레터에도 썸네일 자동 삽입 | ○ | ◎ | 가능 |

→ 네 가지 다 **generate_cardnews.py / generate_newsletter.py 자동화로 처리 가능**
→ 상담 박스·연락처 등 수익화 요소는 **일절 안 넣음** (요청대로)

## 4. 개선 후 블로그 글 예상 모습

```
[제목]
노란봉투법·현대차 파업 ｜ 2026.6.29 오늘의 노동뉴스 5선

[본문]
오늘 노동 뉴스 핵심만 3분 정리 📌

[썸네일 이미지 - 1200×630]

1️⃣ 노란봉투법 시행 후 하청노조 원청 파업 초읽기
   → 원청도 교섭 의무 생김, 대응 필요

2️⃣ 현대차 노조 '순익 30% 성과급' 요구 파업권 확보
   → 임단협 분수령

3️⃣ 금융권 산별중앙교섭 결렬…'임금 6% vs 2.5%'
   → 7월 총파업 가능성

4️⃣ ...
5️⃣ ...

▶ 카드뉴스 전체 보기
https://eu-labornews.vercel.app/...

#노란봉투법 #현대차파업 #성과급전쟁 ...
```

→ 같은 데이터인데 **본문이 꽉 차 보임** = 매력도 ↑ + 검색노출 ↑
