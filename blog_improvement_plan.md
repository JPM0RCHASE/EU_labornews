# 블로그 카드뉴스 매력도 개선 제안

> 20년차 웹디자이너 + 파워블로거 관점에서 분석한 현황과 개선안

---

## 현재 문제 진단

스크린샷을 보면 두 가지가 명확히 보여요.

### 문제 1 — 목록에서 썸네일이 없다
```
[목록 화면]
• 6월 4째주 인사노무 뉴스레터  →  작은 아이콘이라도 있음
• 260622 오늘의 HR 카드뉴스   →  아무것도 없음 (텍스트 URL만 덜렁)
```
블로그 목록에서 **첫 번째 이미지**가 대표 썸네일이 돼요.  
지금은 본문에 이미지가 없으니까 목록에서 회색 빈칸 또는 텍스트만 보여요.  
사람은 0.3초 안에 클릭할지 결정하는데, 이미지 없으면 그냥 스크롤 넘겨요.

### 문제 2 — 제목이 검색/클릭 유입에 취약하다
```
현재:  "260622 오늘의 HR 카드뉴스"
```
- `260622`는 날짜 코드인데 독자 입장에선 의미 없어요
- "HR 카드뉴스"는 카테고리명이지 오늘 내용이 아님
- 네이버·구글 검색에서 이 제목으로 유입될 이유가 없어요

### 문제 3 — 본문이 링크 하나뿐이다
```
현재 본문:
https://eu-labornews.vercel.app/20260622/labornews_20260622.html
```
- 독자에게 "왜 클릭해야 하는지"를 전혀 안 알려줘요
- 검색엔진이 색인할 텍스트 콘텐츠가 없어요
- SNS 공유 시 미리보기만 보이고 블로그 자체 내용은 0이에요

## 개선 방향 3가지

---

### ① 썸네일 이미지 삽입 (가장 효과 큼)

**방법**: 이미 매일 생성되는 **PNG 파일**(`labornews_20260622.png`)을  
블로그 포스트 **본문 맨 위**에 이미지로 넣어야 해요.

```
[개선 후 목록 화면]
• 260622 오늘의 HR 카드뉴스  →  [다크 배경 + "Today's Labor News" 이미지]
```

네이버 블로그·브런치 등 대부분의 플랫폼은  
**본문 첫 번째 이미지 = 대표 썸네일**로 자동 설정해요.

PNG를 자동 업로드하는 방식은 플랫폼마다 API 지원 여부가 달라서  
**단기 대안**: 블로그 포스트 생성 시 PNG 이미지를 본문에 포함시키거나,  
**장기 대안**: 블로그 플랫폼 API(네이버 블로그 API 등)를 통해 자동 업로드.

---

### ② 제목 공식 바꾸기

현재보다 훨씬 클릭하고 싶어지는 제목 공식:

```
[날짜 + 핵심 키워드 2개 + 실용 표현]

예시:
• "최저임금·부당해고 이슈 정리 — 2026.06.22 인사노무 브리핑"
• "오늘의 노동법 핫이슈: 노란봉투법 외 4건 (6/22)"
• "HR 담당자 필독 📌 이번 주 최저임금·산재 이슈 총정리"
```

자동화 포인트: Claude가 카드뉴스 생성할 때 **블로그 포스트 제목도 같이 만들도록** 프롬프트에 추가하면 돼요.

---

### ③ 본문 구조 만들기

링크 하나만 있는 대신, 아래 구조로 본문을 채워요.

```
━━━━━━━━━━━━━━━━━━━━━
📌 오늘의 핵심 이슈 (2026.06.22)

1️⃣ [뉴스 제목 1]
2️⃣ [뉴스 제목 2]
3️⃣ [뉴스 제목 3]

▶ 전체 카드뉴스 보기 → [링크]
━━━━━━━━━━━━━━━━━━━━━
```

이렇게 하면:
- 검색엔진이 색인할 **텍스트 키워드**가 생겨요
- 목록에서 미리보기 텍스트가 의미있게 보여요
- 독자가 클릭 전에 "오늘 뭐가 있는지" 알 수 있어요

## 자동화 가능 여부

현재 `generate_cardnews.py`가 이미 뉴스 5건 + 요약을 갖고 있어요.  
아래 정보를 추가로 출력하도록 스크립트를 수정하면  
**블로그 포스팅 내용 전체를 자동 생성**할 수 있어요:

- 블로그 포스트 제목 (SEO 최적화)
- 본문 텍스트 (3줄 요약 + 링크)
- 대표 이미지 (기존 PNG 활용)

단, **블로그 자동 게시**는 플랫폼마다 다름:
- 네이버 블로그 → Open API 있음 (텍스트만, 이미지 업로드 제한 있음)
- 브런치 → 공식 자동화 API 없음
- 티스토리 → Open API 있음 (이미지 포함 가능)
- 워드프레스 → REST API로 완전 자동화 가능

지금 사용 중인 블로그 플랫폼이 어디냐에 따라 자동화 범위가 결정돼요.
